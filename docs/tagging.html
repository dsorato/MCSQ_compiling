annotation.pos_tagging.output_language_specific_dictionaries(languages)[source]¶

Gets each text segment and its ID from the database, building 4 dictionaries (request, response, instruction, introduction). After, saves the dictionaries as a pickle dump.

Parameters

languages (param1) – list of 3-digit language ISO codes.

annotation.pos_tagging.save_tagged_dictionary(tagger, language)[source]¶

Loads a given untagged dictionary, calls the tagging method and saves the tagged dictionary. :param param1 part-of-speech tagging model: language specific (or multilingual) pos tagging model :type param1 part-of-speech tagging model: Pytorch object :param param2 language: 3-digit language ISO code. :type param2 language: string

annotation.pos_tagging.select_pos_model(language)[source]¶

Selects the appropriate pos tagging model based on the language. ENG language uses a pretrained model provided by Flair. NOR, SPA, GER, CZE, and FRE languages use multilingual pretrained model provided by Flair. CAT, RUS and POR languages use models trained by me.

Parameters

language (param1) – 3-digit language ISO code.

Returns

part-of-speech tagging model (Pytorch object).

annotation.pos_tagging.tag_dictionary(tagger, dictionary)[source]¶

Tags each sentence of the untagged dictionary and updates its value to the tagged sentence.

Parameters

  • part-of-speech tagging model (param1) – language specific (or multilingual) pos tagging model

  • dictionary (param2) – has the IDs and the untagged text.

Returns

A dictionary with the text segment IDs and the tagged text.
